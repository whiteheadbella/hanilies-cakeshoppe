#  Hanilies Cakeshoppe

A Django-based cake and event package ordering system with a custom admin panel.

##  Features

- **Cake Management**: Add, edit, and manage cake products with images
- **Package System**: Create event packages with customizable features
- **Order Tracking**: Real-time order status tracking
- **Payment Integration**: GCash and Cash on Delivery options
- **User Roles**: Role-based access control (Owner, Admin, Manager, Baker, etc.)
- **Custom Admin Panel**: Gold & pink themed admin interface

##  Technology Stack

- **Backend**: Django 6.0.3
- **Database**: PostgreSQL (Production) / SQLite (Development)
- **Frontend**: Bootstrap 5, Font Awesome, Custom CSS
- **Deployment**: Render.com
- **Static Files**: WhiteNoise
- **Image Processing**: Pillow

##  Prerequisites

- Python 3.14 or higher
- pip (Python package manager)
- Git
